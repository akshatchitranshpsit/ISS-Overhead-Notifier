# ISS Overhead Notifier

A Python-based tool that alerts users when the International Space Station is overhead, using real-time API data and location tracking.

## Features
- Tracks ISS position using API data
- Notifies when ISS is visible from your location
- Simple and lightweight

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python iss_notifier.py`
